Entrenamiento de RNA Multicapa con Algoritmo Backpropagation

Descripción

Esta página web permite visualizar el proceso de entrenamiento de una red neuronal artificial (RNA) utilizando el algoritmo de retropropagación (Backpropagation).

Aquí podrás observar cómo la red aprende a partir de datos de entrada y ajusta sus parámetros para obtener una salida precisa.

Visualización

Suba el archivo que contiene los parametros de entrada para mostrar el modelo de la Red Neuronal. Debe ser un archivo .json
